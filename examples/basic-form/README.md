# Component Showcase

A complete GPUI application showcasing all 33 Phase 1-3 shadcn-ui-rs components.

## Components Demonstrated

### Phase 1 -- Form Inputs (12)

1. **Button** - Multiple variants (Default, Outline, Secondary, Destructive)
2. **Input** - Text input with placeholder
3. **Label** - Form labels with required indicator
4. **Checkbox** - Terms acceptance toggle
5. **RadioGroup** - Plan selection (Free / Pro / Enterprise)
6. **Switch** - Notifications on/off toggle
7. **Slider** - Volume control
8. **Select** - Dropdown fruit picker
9. **Toggle** - Bold/Italic text formatting toggles
10. **ToggleGroup** - Alignment selection (Left / Center / Right)
11. **Card** - Form container with header, content, and footer
12. **Dialog** - Modal edit profile dialog

### Phase 2 -- Overlays & Feedback (10)

13. **Alert** - Default and Destructive variants
14. **AlertDialog** - Modal confirmation (shown as static)
15. **Tooltip** - Hover-triggered text overlay
16. **Popover** - Click-triggered overlay (shown as static)
17. **HoverCard** - Hover-triggered card (shown as static)
18. **DropdownMenu** - Click-triggered menu (shown as static)
19. **Sheet** - Slide-in overlay panel (shown as static)
20. **Drawer** - Bottom sheet variant (shown as static)
21. **Toast** - Temporary notification (shown as static)
22. **Sonner** - Stacked toast system (shown as static)

### Phase 3 -- Visual Display (11)

23. **Badge** - Inline status labels (4 variants)
24. **Avatar** - User avatar with fallback initials (3 sizes)
25. **Separator** - Horizontal dividing line
26. **Skeleton** - Loading placeholder (circle + lines)
27. **Progress** - Horizontal progress bars (33% + 66%)
28. **Kbd** - Keyboard shortcut labels
29. **Typography** - H1-H4, Lead, Paragraph, Blockquote, InlineCode
30. **Table** - Data table with header, body, and 3 rows
31. **ScrollArea** - Scrollable container with 10 items
32. **Textarea** - Multi-line text display
33. **Empty** - Empty state placeholder with action button

## Running

```bash
cd examples/basic-form
cargo run
```

This example is a standalone project and is not part of the workspace.
